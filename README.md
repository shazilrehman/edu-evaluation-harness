# Edu Evaluation Harness & Experiment Framework

**LLM-as-Judge** system for scoring and benchmarking AI-generated educational content.

## 🎯 How It Matches LearnWith.AI Requirements
- **Evaluation Sets & Experiment Harnesses**: Curated test cases with expected behavior
- **Quantifiable Success Metrics**: LLM-as-Judge scoring (accuracy, engagement, clarity, etc.)
- **Self-Improving Systems Support**: Results saved for iterative experimentation
- Works together with the Self-Improving Content Generator

## Features
- Automated evaluation on multiple test cases
- Structured JSON scoring + suggestions
- Experiment results saved for analysis
- Streamlit dashboard

## Tech Stack
- LangChain + Ollama
- LangGraph compatible design

This project demonstrates the exact evaluation infrastructure LearnWith.AI needs.
